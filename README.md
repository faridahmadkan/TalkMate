# TalkMate - Telegram AI Bot

Advanced Telegram bot powered by Groq AI.

## Features
- Multiple AI models
- Support ticket system
- User statistics
- Conversation history
- Rate limiting

## Deploy on Render
1. Connect this repository to Render
2. Add environment variables:
   - BOT_TOKEN
   - GROQ_API_KEY
   - ADMIN_IDS
3. Deploy!
